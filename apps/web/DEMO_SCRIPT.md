# Bizflow Demo Script

## 3-5 Minute Walkthrough

### 1. Dashboard Overview (30 sec)

- Open the app at `/dashboard`
- Show revenue, outstanding invoices, and expenses widgets
- Point out the quick action cards

### 2. Receipt to Expense Flow (2 min)

- Click "Open Assistant" or navigate to `/assistant`
- Click the upload button and select any image (or type "Add this receipt as an expense")
- Watch the AI extract receipt data with the tool card animation
- Review the confirmation form with extracted details
- Click "Confirm" to create the expense
- Click "Open expense" to see it in the expenses list

### 4. Data Persistence (30 sec)

- Navigate to Expenses and Invoices pages
- Show that newly created items persist (localStorage)
- Switch between DE/EN languages
- Toggle dark/light theme

### 5. ERP-Ready Architecture (30 sec)

- Show the sidebar with "Coming Soon" modules
- Explain the modular architecture for future expansion

## Key Features to Highlight

- Tool-based AI assistant with streaming responses
- Multi-language support (DE/EN)
- Dark-first design with theme toggle
- German freelancer domain (EUR, VAT rates, invoice numbering)
- Responsive design for mobile/desktop
